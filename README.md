### **swap a / swap b:**  
swap first two elements at the top of choosen stack  
**A:** `1` `5` 3 4 2

**A:**
`1` `5`2 4 3

| A | A' |
|:----:|:----:|
| `3` | `2` |
| `2` | `3` | 
| 1 | 1 |

### **pa** push a - take the first element at the top of b and put it at the top of a
|    A | B   | | A    | B   |
|:----:|:----:|-|:----:|:----:|
| 3 | `4` | | `4` | 5 |
| 2 | 5 | | 3 | 6 | 
|   | 6   | | 4 |
