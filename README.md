# Push_swap

## Operations

### **sa/sb:** swap first two elements at the top of choosen stack
| A | A' |
|:----:|:----:|
| `3` | `2` |
| `2` | `3` | 
| 1 | 1 |

### **ss** 
|    A | A'   | | B    | B'   |
|:----:|:----:|-|:----:|:----:|
| `3` | `2` | | `5` | `6` |
| `2` | `3` | | `6` | `5` | 
| 1   | 1   | | 4 | 4 |
